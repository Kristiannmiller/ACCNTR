# PromoCarousel

Story Link: [#77]('https://github.com/ahc-digital/migration-planning-tracking/issues/77')

Description: The promo carousel is a carousel component that is made up of promo callouts. The presentation of each slide matches that of a promo callout with the addition of the carousel controls. The carousel is populated by adding multiple promo callouts to a placeholder that then builds a carousel from them.

---
# Acceptance Criteria
The component is created within Sitecore and meets the following criteria:

Template name and structure matches specification in "Sitecore Details" section
Content authors are able to add, modify, and remove the component via experience editor
For ADA, component should be fully operable via keyboard (tab index, panel toggling, etc) without mouse interaction. Component must also pass Storybook ADA plugin validation.

- Component is added to storybook, and each content author editable field is editable within storybook.

- In Experience Editor, each slide is stacked on top of each other for ease of editing.

- Content authors are able to edit the slide timing (default is 6 seconds).

- Content authors are able to specify a background at the carousel level. This background overrides the background of all individual slides.

- Content authors are able to specify image side at the carousel level. This option overrides the image side selection of all individual slides.
