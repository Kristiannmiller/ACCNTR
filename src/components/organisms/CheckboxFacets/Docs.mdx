# CheckboxFacets

Story Link: [460](https://github.com/ahc-digital/migration-planning-tracking/issues/460)

Description: The CheckboxFacets component is used for the ElasticSearch UI to allow users to select one or more options from a list of options.

---
# Acceptance Criteria

- Accepts a list of values and creates a checkbox for each
- Expand/Collapse functionality is applied on mobile
