# SidePhotoContentCard

Story Link: [#164]('https://github.com/ahc-digital/component-library/issues/164')

The Side Photo Content Card is used to display a single image alongside a featured link or CTA.

---
# Acceptance Criteria

Side Photo Card Rendering

**Fields:**

Headline - Single Line Text [Required]

Support Copy - Rich Text [Optional]

CTA button - General Link [Required]

Secondary CTA button - General Link [Optional]

**Rendering Parameters:**

Title type: H2, H3, Paragraph Bold [Default]

Content layout: 50/50 [Default], 60/40

Image side: Left [Default], Right

Secondary CTA style: Outlined, Link, Solid [Default]

Corporate Theme Needed? Yes
