# SideStatistics

Story Link: [#206]('https://github.com/ahc-digital/migration-planning-tracking/issues/206')

The side by side statistics is a single column variant of the statistics.

---
# Acceptance Criteria

**Side Statistics Rendering**

Rendering Name: C51 - Side Statistics

Template Name: C51 - Side Statistics

Allowed Renderings: C51 - Side Statistics

**Fields**
  - Headline - Single Line Text [Optional]
  - Support copy - Rich Text [Optional]
  - Should use Content Author rich text profile
  - Support copy bottom - Rich Text [Optional]
  - Should use Content Author rich text profile
  - CTA - General Link [Optional]

**Rendering Parameters**
  - Title type - Droplink
    - H2
    - H3
    - Paragraph bold [Default]
  - Content alignments - Droplink
    - Center [Default]
    - Left
  - Background Color - Droplink
    - White [Default]
    - White Smoke
    - Denim Blue

The component is created within Sitecore and meets the following criteria:

Template name and structure matches specification in "Sitecore Details" section
Content authors are able to add, modify, and remove the component via experience editor
For ADA, component should be fully operable via keyboard (tab index, panel toggling, etc) without mouse interaction. Component must also pass Storybook ADA plugin validation.

Component is added to storybook, and each content author editable field is editable within storybook.

Should implement the exclude from translation class around the entire component.

The top of the container should be the anchor target with the id from the rendering parameter.

The alignment between the statistic and support copy for each item is baseline aligned.
