# BioCardDirectory

Story Link: [#105](https://github.com/ahc-digital/migration-planning-tracking/issues/105)

Description: The bio cards displays a list of biography data pulled from biography page items. The cards come in two variants: directory and grid. The directory displays detailed information including Name, Title, Specialty, Phone, Fax, Email, Locations and Additional Information.

---

# Acceptance Criteria

The component is created within Sitecore and meets the following criteria:

-   Template name and structure matches specification in "Sitecore Details" section
-   Content authors are able to add, modify, and remove the component via experience editor

For ADA, component should be fully operable via keyboard (tab index, panel toggling, etc) without mouse interaction. Component must also pass Storybook ADA plugin validation.

Component is added to storybook, and each content author editable field is editable within storybook.

Each bio card is displayed in a single column with the listed information.

If a particularly field is not filled out for a bio card, nothing should be displayed on that list entry. (No label or field value)
