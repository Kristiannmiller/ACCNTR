# LocationInfo

Story Link: [#227]('https://github.com/ahc-digital/migration-planning-tracking/issues/227')

Description: The Location Info Block displays detailed information about a location data template. This may or may not include a map, as well as information such as location name, hours, etc.

---
# Acceptance Criteria
The component is created within Sitecore and meets the following criteria:

Template name and structure matches specification in "Sitecore Details" section
Content authors are able to add, modify, and remove the component via experience editor
For ADA, component should be fully operable via keyboard (tab index, panel toggling, etc) without mouse interaction. Component must also pass Storybook ADA plugin validation.

Component is added to storybook, and each content author editable field is editable within storybook.

Should implement the exclude from translation class around the entire component.

All content should be left aligned.

All info should be pulled automatically from the linked location object. No data fields should exist on the rendering template it itself.

Multiple columns of locations will be handled via the multi column container rendering. When a location info is placed inside this rendering, the map should hide automatically.
