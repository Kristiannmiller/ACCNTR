# HeroStatisticsItem

Story Link: [#204]('https://github.com/ahc-digital/migration-planning-tracking/issues/204')

Description: The HeroStatisticsItem is to be used as a child element to the HeroStatistics component. 
It is used to display a single statistic with a label and value.
