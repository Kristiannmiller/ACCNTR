# ReverseRow

Story Link: [#135]('https://github.com/ahc-digital/migration-planning-tracking/issues/135')

Description: The Reverse Row Story is a series of content blocks with an alternating layout. Each block consists of an image or video with accompanying text.

---

# Acceptance Criteria

- For ADA, component should be fully operable via keyboard (tab index, panel toggling, etc) without mouse interaction. Component must also pass Storybook ADA plugin validation.

- Component is added to storybook, and each content author editable field is editable within storybook.

- Content authors are able to add an image or video. If both are filled out, the video should take precedence over the image.

- Content authors are able to specify placement of the image/video (left or right).

- Content authors are able to remove the padding between rows.

- On mobile, the image will always come before content (image stacked on top).
