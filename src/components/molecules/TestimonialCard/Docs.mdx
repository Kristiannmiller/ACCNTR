# TestimonialCard

Story Link: [#273](https://github.com/ahc-digital/migration-planning-tracking/issues/273)

Description: The TestimonialCard component is being used in the Testimonials component. Each card contains a singular quote.

---
# Acceptance Criteria
The component is created within Sitecore and meets the following criteria:

-  Template name and structure matches specification in "Sitecore Details" section
-  Content authors are able to add, modify, and remove the component via experience editor

For ADA, component should be fully operable via keyboard (tab index, panel toggling, etc) without mouse interaction. Component must also pass Storybook ADA plugin validation.

Component is added to storybook, and each content author editable field is editable within storybook.

Should implement the exclude from translation class around the entire component.

All testimonial cards should be sized to the height of the tallest card in the row.

If the 'Show First Row Only on Load' rendering parameter is checked, only the first row of testimonials is show and the rest of the cards are hidden behind a "Show more" CTA functionality. Once the content has been expanded, the "Show More" CTA should disappear.

The testimonial quote should be italic, and the name of the individual providing the quote should be bold.

The quote should be marked as a quote in the HTML markup for accessibility considerations.
