# Sheet

Description: The Sheet component is used to overlay content on top of the page. It's used primarily for the global mobile navigation menu.

---

## Example Usage(s)
```tsx
<Sheet>
    <SheetTrigger>Open</SheetTrigger>
    <SheetContent direction='bottom'>
        <SheetHeader>
            <SheetTitle>Hello!</SheetTitle>
            <SheetDescription>
                This is an example of a sheet sliding in from the
                bottom.
            </SheetDescription>
        </SheetHeader>
    </SheetContent>
</Sheet>
```
