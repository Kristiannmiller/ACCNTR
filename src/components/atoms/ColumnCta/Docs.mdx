# ColumnCta

Story Link: [#133]('https://github.com/ahc-digital/migration-planning-tracking/issues/133')

Description: 

The ColumnCta component is used for creating columns to be used in the MultiColumnCta. It contains a title, a rich text block and a cta link all optionally. 

---
# Acceptance Criteria
Each column consists of:
 - Headline (h2/h3/P-bold)
       - Center of left align (default left)
 - Rich text field
 - Link list
      - Center of left align (default left)
 - Optional CTA button

<Stories />

## Props
<Controls />

- headline [Required]
  - Text for the headline

- headlineTag
  - Allows you to choose p, h1 or h2 tag for the headline. Default is p.

- bodyCopy [Required]
  - body copy source

- textAlignment
  - Allows you to change text alignment. Default is left.

- extraCTA [Optional]
  - CTA button link
