# BackgroundColor

Description: The BackgroundColor component is a Slot that sets the background color of the children.
It also creates a ReactContext so child components can swap styles based on the background color.

---
# Acceptance Criteria
Background Colors:
- 'white'
- 'whiteSmoke'
- 'wellnessBlue'
- 'denimBlue'
- 'achOrange'
- 'cchaPurple'
