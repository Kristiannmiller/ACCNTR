# DropdownMenu

Story Link: [#105]('https://github.com/ahc-digital/component-library/issues/105')

Description: The DropdownMenu component is used as the base atom for the left navigation and left jump navigation

---
# Acceptance Criteria

The component is created within Sitecore and meets the following criteria:

Template name and structure matches specification in "Sitecore Details" section
Content authors are able to add, modify, and remove the component via experience editor
For ADA, component should be fully operable via keyboard (tab index, panel toggling, etc) without mouse interaction. Component must also pass Storybook ADA plugin validation.

Component is added to storybook, and each content author editable field is editable within storybook.

"Back to Parent" link is always the first link in the list.

Sections with child pages can be expanded and collapsed. Only one section can be expanded at a time.

If more than 10 links are present, a "Show More" link should appear at the bottom of the list and display all links past the 10th once clicked. If the list is expanded, a "Show Less" link should be shown to collapse the list.

On mobile the entire list should be collapsed by default.

Content authors can control the text that appears for each page via the "Navigation Title" field on the page item.

Content authors can control the root of the left navigation using via the "Is Site Root" field on the page item.

Content authors can exclude an item from the left navigation via a field on the page item.

Pages of type "Link Page" should render out the link field on the page item instead of generating a link directly to page.

## Example Usage(s)
```tsx
 <DropdownMenu type={'multiple'}>
    <DropdownMenuItem value='item-1'>
        <DropdownMenuTrigger>
            <span>Page Navigation </span>
            <SvgIcon icon='bars' />
            <SvgIcon icon='close' />
        </DropdownMenuTrigger>
        <DropdownMenuContent>
            <DropdownMenuSeparator>
                <SvgIcon icon='angleLeft' />
                <Anchor href='https://www.advocatehealth.com'>
                    Top level
                </Anchor>
            </DropdownMenuSeparator>
            <DropdownMenuList>
                <DropdownMenuListItem>
                    <Anchor href='https://www.advocatehealth.com'>
                        Link 1
                    </Anchor>
                </DropdownMenuListItem>
                <DropdownMenuListItem>
                    <Anchor href='https://www.advocatehealth.com'>
                        Link 2
                    </Anchor>
                </DropdownMenuListItem>
                <DropdownMenuItem value='item-2'>
                    <DropdownMenuListItem>
                        <div>
                            <Anchor href='https://www.aurorahealthcare.org/'>
                                Expandable section
                            </Anchor>
                            <DropdownMenuSubTrigger />
                        </div>
                        <DropdownMenuContent>
                            <DropdownMenuList>
                                <DropdownMenuListItem>
                                    <Anchor href='https://www.advocatehealth.com'>
                                        Sub Link 1
                                    </Anchor>
                                </DropdownMenuListItem>
                                <DropdownMenuListItem>
                                    <Anchor href='https://www.advocatehealth.com'>
                                        Sub Link 2
                                    </Anchor>
                                </DropdownMenuListItem>
                                <DropdownMenuItem value='item-3'>
                                    <DropdownMenuListItem>
                                        <div>
                                            <Anchor href='https://www.aurorahealthcare.org/'>
                                                Expandable section
                                            </Anchor>
                                            <DropdownMenuSubTrigger />
                                        </div>
                                        <DropdownMenuContent>
                                            <DropdownMenuList>
                                                <DropdownMenuListItem>
                                                    <Anchor href='https://www.advocatehealth.com'>
                                                        Sub Link 1
                                                    </Anchor>
                                                </DropdownMenuListItem>
                                                <DropdownMenuListItem>
                                                    <Anchor href='https://www.advocatehealth.com'>
                                                        Sub Link 2
                                                    </Anchor>
                                                </DropdownMenuListItem>
                                            </DropdownMenuList>
                                        </DropdownMenuContent>
                                    </DropdownMenuListItem>
                                </DropdownMenuItem>
                            </DropdownMenuList>
                        </DropdownMenuContent>
                    </DropdownMenuListItem>
                </DropdownMenuItem>
            </DropdownMenuList>
        </DropdownMenuContent>
    </DropdownMenuItem>
</DropdownMenu>
```
