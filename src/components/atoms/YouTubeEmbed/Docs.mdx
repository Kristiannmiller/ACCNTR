# YouTubeEmbed

Description: The YouTubeEmbed component is used for embedding YouTube videos in an iframe element. It accepts a YouTube videoId.

---
# Acceptance Criteria
- Accepts a YouTube `videoId`
  - type: `ReactNode`
- Renders iFrame of corresponding YouTube video
- Domain must be `https://www.youtube-nocookie.com/embed/`
