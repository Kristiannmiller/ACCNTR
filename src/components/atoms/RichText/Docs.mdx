# RichText

Story Link: [#83]('https://github.com/ahc-digital/component-library/issues/83')

# Acceptance Criteria
Use @extend to extend the styles applied in Text to their corresponding elements nested under the class rich-text.

### Sitecore Implementation Example:

```tsx
const mockSitecoreRichText = {
    value: '<p><a href="https://www.aurorahealthcare.org" target="_blank">Lorem ipsum dolor</a> sit amet, consectetur adipiscing elit. Nunc quis turpis </p>'
};
<RichText>
    <JSSImage field={mockSitecoreRichText} />
</RichText>;
```
