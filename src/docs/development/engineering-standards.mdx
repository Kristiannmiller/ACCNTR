# Engineering Standards
The following standards are intended to provide guidance and best practices for Advocate Health developers.
These standards are intended to be used as a reference and should be followed when developing software for Advocate Health.

These standards are maintained in the [Advocate Health Engineering Standards Wiki](https://github.com/ahc-digital/engineering-standards/wiki#advocate-health-engineering-standards).
