# Contributing Guide

## Introduction
Thank you for considering contributing to the React Component Library! This guide will help you get started with the contribution process.

Our team works best by creating tickets into the backlog before starting work. This helps us to prioritize and track
the work that needs to be done. Please create a ticket in the backlog for a feature request, bug report, or any other issue.



## FAQs
<details>
    <summary><strong>Where can I create a ticket in the backlog?</strong></summary>
    <p>Go to the issues tab of the [Component Library](https://github.com/ahc-digital/component-library/issues/new/choose) and add details.
        Make sure to tag one of the [core contributors](/docs/foundation-introduction--docs#maintainers-and-resources) to the issue.</p>
</details>
